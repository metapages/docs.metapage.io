Connect the metapage file system

Each metapage has it’s own connected file system, where you can

  • upload files
  • download files
  • save metaframe outputs as files
  • use files as metaframe inputs

Example​

The example below shows the “files” metaframe saving the generated plot output files.

Walkthough: how to add and connect the files metaframe

Using the example metapage below, we will connect the file system, and download the generated plot images.

note

Skip this step if you already have a metapage: go to “2. Edit Mode”

1. Clone an example metapage​

Click Share and Clone :

2. Edit mode​

3. Add the Files frame​

4. Connect the Files frame​

Make it bigger and go back into View mode

5. Download the images generated!​

Select the files and a menu appears, allowing you to delete, download, or send the files downstream.

Congratulations​

Now you can upload, download, and save files in your workflows.

Extra Credit​

File system metaframes from one metapage can be used in another metapage

This is one way to connect metapage inputs and outputs to other metapages.

1. Copy the File System metaframe URL​

2. Go to another metapage, and add the URL​

Paste the URL from above into the dialog.

Now files from one workflow/metapage can be used by other metapages, allowing you to easily connect different workflows by the files produced.
