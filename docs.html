Metapages are workflows that run in the browser.

They solve the problems of connecting code, data, and compute that can be easily shared, stored, and modified.

Metapages remove the friction of communication, allowing instant and rich communication and collaboration between scientists, data scientists, and engineers.

They allow the use of, and data connections between:

  • any open source programming languages
    • Python
    • R
    • Julia
    • Go
    • Octave
    • Java
    • Anything that can be docker containerized
  • any front end tools and visualization frameworks

all within a web page that can be embedded and published in almost any medium.

A metapage is a set of websites connected together with data pipes. Metapages are hybrid apps, workflows, dashboards.

You can create, share, embed, and customize at metapage.io

Problems they solve:

  • Velocity: accelerate web development by embedding entire workflows or tools
  • Dashboards: create powerful dashboards, connecting any web app, library, or data source
  • Collaboration: scientific and data-heavy workflows can be shared and worked on collaboratively
  • Data Science / Machine Learning / AI: Rapidly iterate, develop, test and even deploy machine learning code, models, and pipelines
  • Publish: share your app, code, or data for others to instantly and easily (no-code) integrate with their app/tools/website
